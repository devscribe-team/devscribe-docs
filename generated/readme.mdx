## devscribe-frontend

### Troubleshooting

<Callout type="warning">
**Google / GitHub OAuth doesn't work**

Ensure that `NEXT_PUBLIC_DEVSCRIBE_BACKEND_URL=http://127.0.0.1:8000` in your `.env` file.

**Important:** The URL must be exactly `http://127.0.0.1:8000` (not `localhost`). The OAuth implementation is pedantic about this specific format.
</Callout>
